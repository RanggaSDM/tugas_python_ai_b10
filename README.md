# Tugas Python AI B10
Kumpulan tugas pemrograman Python untuk struktur data, fungsi, class, dan analisis data.

## Ringkasan Tugas:
- **Tugas 4:** Implementasi List, Tuple, Set, dan Dictionary.
- **Tugas 5:** Dasar-dasar Function dan Object-Oriented Programming (OOP) sederhana.
- **Tugas 6:** Analisis data menggunakan NumPy & Pandas, pengelolaan File I/O, dan OOP tingkat lanjut.
